Termos de Informática

css
Cascading Style Sheets e resposavel por definir a aparência de elementos HTML.
JavaScript
Linguagem de programação usada para criar interatividade em páginas web.
HTML
Linguagem de marcação usada para criar páginas web.
Github
Plataforma de hospedagem de código fonte e controle de versão.
Git
Sistema de controle de versão distribuído.
